This is a project I’ve been working on as I slowly learn Rust. There's no progression and all stats are fixed.

Controls:

Main Menu - use arrow keys to select, and enter to confirm.
Quit - doesn’t do anything in this browser-version except break the game
Likewise, new game only works the first time. I’ve got some runstate routing issue I need to fix. If you want a new game without dying first, then simply reload the page.

Movement

  • Cardinal directions - Up/Down/Left/Right - J/K/H/L or:
    • Numpad 8/2/4/6
  • Diagonal movement - Y/U/N/M or:
    • numpad 7/9/1/3
  • ’>’ or period key to travel down stairs (also depicted in-game by the ‘>’ symbol)

Interaction
Mouseover objects in the game world to see a small tooltip.
Target enemies with ranged items using the mouse, and left-click to use

  • G - Pick up item under the player
  • I - Inventory
  • D - Drop item
  • Spacebar/Numpad5 - Wait a turn
